# 🎮 PCL整合包EXE生成器

## 📋 项目简介

这是一个专为新手设计的**PCL整合包安装器生成工具**，让你只需三步就能轻松创建属于自己的整合包安装程序！

### ✨ 核心功能

- **一键生成**：简单三步，生成可独立运行的EXE文件
- **无需编程**：友好的图形界面，无需任何编程知识
- **自动打包**：整合包自动嵌入EXE，无需额外文件
- **即生成即用**：生成的EXE可以直接分发给其他玩家
- **智能安装**：自动检测并安装整合包到PCL启动器

## 🛠️ 系统要求

- **操作系统**：Windows 7/8/10/11
- **硬件要求**：至少2GB内存，500MB可用磁盘空间

## 📦 安装说明

### 方法1：直接使用（推荐）

1. 下载项目压缩包并解压
2. 双击运行 `启动生成器.bat` 文件
3. 等待程序启动，开始使用！

### 方法2：手动启动（需要**python环境**）

1. 打开命令提示符
2. 进入项目目录
3. 输入 `python -m src.main` 并回车

### 制作 `EXE一键启动程序` 部分

#### 准备工作

1. 确保你已经成功导出了PCL整合包文件（.mrpack格式）
2. 确保你已经下载并解压了本项目压缩包
3. 确保你的电脑满足系统要求

#### 第1步：启动EXE生成器

1. 找到解压后的项目文件夹
2. 双击运行 `启动生成器.bat` 文件
3. 等待程序启动，你会看到一个友好的图形界面
4. 界面上方会显示 `🎮 PCL整合包EXE生成器 🎮` 标题

#### 第2步：选择整合包文件

1. 找到界面中 `📦 1. 选择您的整合包文件` 区域
2. 点击右侧的 `浏览...` 按钮
3. 此时会弹出一个文件选择窗口
4. 在文件选择窗口中，导航到你之前导出的整合包文件所在的文件夹
5. 找到后缀为 `.mrpack` 的整合包文件
6. 点击选中该文件
7. 然后点击窗口右下角的 `打开` 按钮
8. 此时你会看到文件路径显示在左侧的输入框中

#### 第3步：设置输出EXE名称

1. 找到界面中 `🏷️ 2. 设置输出EXE名称` 区域
2. 在 `文件名:` 输入框中，为你的安装器取一个好听的名字
3. 例如：`我的模组包安装器`、`MC天堂整合包` 等
4. 注意事项：
   - 不要包含特殊字符（如：`!@#$%^&*()`）
   - 不要使用过长的名称（建议不超过20个字符）
   - 尽量使用中文或英文名称

#### 第4步：生成EXE文件

1. 确认前面的步骤都已完成：
   - ✅ 已选择整合包文件
   - ✅ 已设置输出名称
2. 找到界面下方的 `🚀 3. 生成EXE文件` 按钮
3. 点击该按钮开始生成
4. 此时界面底部的状态会显示：`正在准备整合包文件...`
5. 接下来会显示：`正在生成EXE文件...`
6. 生成过程需要1-2分钟，请耐心等待
7. 生成成功后，会弹出一个提示窗口：`EXE文件生成成功！输出路径: dist/xxx.exe`
8. 点击 `确定` 关闭提示窗口

#### 第5步：找到并使用生成的EXE文件

1. 返回到项目文件夹
2. 找到名为 `dist` 的文件夹
3. 打开该文件夹
4. 你会看到一个 `.exe` 文件，这就是你刚刚生成的整合包安装器
5. 你可以：
   - 直接双击运行测试效果
   - 复制该文件到桌面，方便后续使用
   - 将该文件通过QQ、微信等方式分享给其他玩家

## 🎯 玩家使用方法（分享给其他玩家）

当其他玩家收到你生成的EXE文件后，他们可以按照以下步骤使用：

1. 双击运行你分享的EXE文件
2. 此时会弹出安装器界面
3. 点击 `浏览...` 按钮，选择PCL启动器所在的目录
4. 确认目录选择正确
5. 点击 `一键安装并启动PCL` 按钮
6. 程序会自动：
   - 检查目录是否正确
   - 关闭正在运行的PCL进程
   - 将整合包文件复制到正确位置
   - 启动PCL启动器
7. 安装完成后，PCL会自动启动，并开始导入整合包

## ❓ 常见问题

### Q1：双击 `启动生成器.bat` 没有反应怎么办？

**A1**：
- 检查是否安装了Python环境
- 检查Python是否已添加到系统PATH
- 尝试使用方法2手动启动
- 检查项目文件夹是否完整

### Q2：生成EXE时出现错误怎么办？

**A2**：请检查：
- 你的整合包文件是否完整，没有损坏
- 输出文件名是否包含特殊字符
- 确保你有足够的磁盘空间（至少500MB）
- 确保PCL整合包文件格式正确（.mrpack）

### Q3：生成的EXE文件很大，正常吗？

**A3**：正常！因为EXE文件包含了：
- 完整的Python运行环境
- 你的整合包文件
- 安装器程序

通常大小在10-20MB之间，如果整合包本身很大，EXE也会相应变大。

### Q4：生成的EXE可以在其他电脑上运行吗？

**A4**：当然可以！生成的EXE是完全独立的，无需在其他电脑上安装任何依赖。

### Q5：如何更新我的整合包？

**A5**：
1. 在PCL中重新导出更新后的整合包
2. 使用本工具重新生成一次EXE文件
3. 用新生成的EXE替换旧版本即可

### Q6：支持哪些整合包格式？

**A6**：目前仅支持 `.mrpack` 格式的整合包，这是PCL官方推荐的标准格式。

### Q7：生成的EXE会报毒吗？

**A7**：
- 本工具生成的EXE是安全的，不会包含病毒
- 由于是新生成的程序，部分杀毒软件可能会误报
- 建议将生成的EXE添加到杀毒软件的白名单中
